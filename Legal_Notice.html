26.2.2020

Have had issues with complying with the terms of CC BY-SA 3.0 licence. The first part of these issues, are discussed here.

To address the first part of these issues, I've decided to download all the lists of contributors for every used page that is mentioned in the section dedicated to the material licensed under CC BY-SA 3.0. Then with those lists, I'll insert them into the section just mentioned, with the lists organised so that each list is displayed with the page for which the list is the contributor list. Each contributor name in these lists, will also be a hyperlink to the Wikipedia user page for the contributor. This decision for addressing the first part of these issues, should provide sufficient credit, with the required level of prominence, with respect to complying with the CC BY-SA 3.0 licence. By following through with this decision, the first part of these issues will be completely addressed.

Example illustrating procedure to be used to get contributor lists at least for Wikipedia material (a similar procedure might also exist for Wikishire material)

To get the list of contributers to the Wikipedia Scotland page, first run:
    https://en.wikipedia.org/w/api.php?action=query&prop=contributors&titles=Scotland&pclimit=500
Then keep on running :
    https://en.wikipedia.org/w/api.php?action=query&prop=contributors&titles=Scotland&pclimit=500&pccontinue=??????
where ?????? is the pccontinue value returned by the previous GET request, until no pccontinue value is returned (also an empty batchcomplete value will also likely be returned at such time).


See: 
    https://www.mediawiki.org/wiki/API:Main_page
            and 
    https://www.mediawiki.org/wiki/API:Contributors
for more information.


Unix command `wget` can be used for running the URLs and downloading the responses, but characters will have to be escaped accordingly.


On 27.2.2020, a script was created to retrieve such a contributor list for any particular Wikipedia title. The script was stored in the GitHub repository at this location.
