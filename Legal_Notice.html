Concerning the Northern Ireland data of the tables that end with the text 'ToCounties', data was created using:

  • the Batch Coordinate Converter tool authored by Barry Hunter
  • the GridInQuest II Coordinate Transformation Utility Main Unit tool [Copyright (C) 2015 Paul Michell, Michell Computing]
    What more can you tell me about this tool and how it was used?
    The author of the tool has said that the tool output is not part of the copyright of the program, and no terms are imposed on simply using the program. The tool output was generated by transforming the ETRS89 Geodetic coordinates for entries in the OS Northern Ireland (NI) Open Data Placename Gazetteer product, to OSGB36 (cartesian) coordinates. The output was then plugged into the Wikishire lookup tool to correspond the places in the gazetteer product to the historic counties of Northern Ireland. The correspondence between the place names in the gazetteer and the historic counties discovered for these places, is what is stored in the data (output coordinates are not stored).
